# 🌈 Color Changer

This project allows users to change the background color of a webpage by clicking buttons. 🎨

## 🛠️ Technologies Used

- JavaScript
- HTML
- CSS

## 🚀 How to Use

1. Clone the repository. 📂
2. Open `index.html` in your browser. 🌐
3. Click the buttons to change the background color. 🎨
